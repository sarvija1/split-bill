# split-bill

## Prerequisites

### Yolo

Install [node.js](https://nodejs.org/en/download/) 14.4.0 or later globally

In project root:

    npm install

### Using nvm

Install [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

In project root:

    nvm use && npm install

## Running

In project root:

    npm start

## Modify input data

Edit entries in [data.js](/src/data.js)
